CSS 盒模型

基本概念

盒模型又称框模型（Box Model）,包含了内容（content）、内边距（padding）、边框（border）、外边距（margin）4个要素。

如图：

两种盒模型的区别

两种盒模型分别是

  • content-box 内容盒，内容是盒子的边界
  • border-box 边框盒，边框是盒子的边界

可以通过设置 box-sizing 属性来设置盒模型类型

宽度计算方式不同

  • content-box width = 内容宽度
  • border-box width = 内容宽度 + padding + border

拓展：外边距重叠

哪些情况重叠

  • 父子 margin 合并
  • 兄弟 margin 合并

如何阻止重叠

  • 父子合并用 padding / border 挡住
  • 父子合并用 overflow: hidden 挡住
  • 兄弟合并可以用 display:inline-block 消除
